What is ibuntu?

ibuntu is a custom Linux distro with one goal: a fast, clean system with no cruft. If you've ever reformatted a fresh install on day one to remove the garbage, ibuntu is for you.

Who built it?

An independent project maintained by one developer. It started as a personal scratch-your-own-itch experiment and grew into something worth shipping to the community.

Contributing

PRs, bug reports, and feature requests are all welcome. Open an issue before starting big work. The repo is on GitHub — clone it, break it, fix it.

Roadmap

Planned: graphical package manager, ARM/Raspberry Pi builds, improved hardware detection, and a minimal server edition. Watch the repo for updates.
