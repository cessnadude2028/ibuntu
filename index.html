What is ibuntu?

ibuntu is a custom Linux distribution built from the ground up with a specific vision: a system that's fast, clean, and genuinely pleasant to use from day one. No cruft, no ads, no compromises.

Who built it?

ibuntu is an independent project maintained by its creator. It started as a personal experiment and grew into something worth sharing with the wider Linux community.

Contributing

ibuntu is open source. Bug reports, feature suggestions, and pull requests are all welcome on GitHub. Check the issues page to see where you can help.

Roadmap

Upcoming work includes a graphical software centre, improved theming support, and ARM builds for Raspberry Pi. Follow the repo for updates.

ibuntu is free and open source software, released under the GPL. All source code is available on GitHub.
